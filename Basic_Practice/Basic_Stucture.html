The HTTP header User-Agent is a request header that sends a characteristic string to web servers,
allowing them to identify the Operating System (OS) and browser of the client making the request. Every time your browser connects to a website, it includes the User-Agent field in the HTTP header.
This information helps websites serve content tailored to different browsers and operating systems. Since different browsers use varying formats, there is no standardized way of writing a User-Agent string.
Furthermore, modern browsers often add significant amounts of information to the User-Agent field.
